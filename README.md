# Handwritten Digit Recognition

A CNN-based handwritten digit recognition system built using Python, PyTorch, and Tkinter.

## Features
- Train CNN on MNIST dataset
- GUI for drawing digits
- Real-time prediction
- Model saved as .pth

## Technologies
- Python
- PyTorch
- Tkinter
- NumPy
- Pillow

## Run

```bash
python gui_cnn.py
